# Excel Word Lists for Genki Study Resources

This folder contains vocabulary lists for Genki 3rd Edition in the [lessons](lessons/) folder.

### How to Download the Lists

To download a single list, you can go to an edition's folder, click the list, and then download it via the download button.

To download all lists at once, you can [download this repo](https://github.com/tannertanner1/studyresources), unzip the contents, then navigate to **resources/tools/wordlist_E-J** and open the folder for the 3rd Edition (lessons) to access their lists.
